### API Routes (src/app/api/)

POST /api/ingest
- Accept CSV upload (multipart) OR trigger from scripts/data/synthetic_150k.csv
- Run de-identification pipeline in memory:
  - DROP: first_name, last_name, ssn, street (exact), phone, email (replaced by tokens)
  - HASH: email -> email_token (SHA256 + salt), phone -> phone_token
  - TRANSFORM: dob->age_range+life_stage+generation, zip->zip3+metro+urban_class, income->income_range+tier, employer->industry+tier+stability, fico->band+profile, dti->bucket, ltv->bucket, loan_amount->bucket
- Generate two text templates per record (Profile + Intent)
- Embed both via OpenAI text-embedding-3-small
- Upsert to Pinecone with metadata
- Delete source
- Return tranche manifest

GET/POST /api/query
- Accept natural language query OR structured filters
- Embed query via OpenAI
- Search Pinecone by cosine similarity
- Return top N results with metadata (no tokens)

POST /api/score
- Accept a new lead (raw fields)
- De-ID it in memory
- Embed it
- Find 20 nearest neighbors in Pinecone
- Return propensity score (funded neighbors / total neighbors * 100)

GET /api/segments
- Fetch sample of vectors from Pinecone
- Run k-means clustering (k=8)
- Label each cluster by dominant attributes
- Return segment profiles

GET /api/export
- Accept segment filter or query
- Return matching email_token + phone_token pairs
- Format: CSV download ready for Facebook/Google Custom Audience upload

GET /api/status
- Return Chroma collection stats / counts when possible
- Return tranche manifest list from a simple JSON file

### Frontend (src/app/)

Main page: Dashboard at / with sidebar nav and 6 tabs:

1. Ingest tab - Upload CSV or click 'Run on synthetic dataset'. Show progress bar, records processed, time elapsed, final manifest.
2. Query tab - Natural language search box. Show results as cards with all signal fields. No PII.
3. Score tab - Form to enter a new lead's details. Show score 0-100, nearest neighbor profiles.
4. Segments tab - Button to run clustering. Show 8 segment cards with dominant attributes, record count, funded rate.
5. Export tab - Filter by segment or query. Show count. Download as CSV (email_token, phone_token only).
6. Status tab - Show Chroma collection stats, total vectors, tranche history table.
